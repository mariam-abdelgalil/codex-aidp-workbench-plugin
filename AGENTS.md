# Oracle AIDP Workbench Plugin

This plugin provides Codex skills for Oracle AI Data Platform Workbench tasks:
catalog discovery, Spark SQL, Delta Lake DDL/DML, ingestion, pipelines,
clusters, governance, MLOps, agent flows, and workspace operations.

When a user asks about AIDP Workbench tasks, read the relevant `SKILL.md` file
under `skills/` before acting and follow its instructions.

## Local AIDP Connection Details

Do not commit real values here in a public fork. Replace these placeholders in
your local working copy or provide them in your own private workspace
instructions.

- AIDP Instance OCID: `<your-aidp-instance-ocid>`
- Compartment OCID: `<your-compartment-ocid>`
- Region: `<your-oci-region>`
- Workspace key or ID: `<your-workspace-key-or-id>`

Use the official `aidp` CLI where available. The skills document `oci
raw-request` fallbacks for operations where the CLI is unavailable.
